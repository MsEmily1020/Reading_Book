# Icon 가운데 정렬

```dart
MaterialApp(
  home : Scaffold(
    body: Row(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Icon(Icons.star),
        Icon(Icons.star),
        Icon(Icons.star),
      ],
    ),
  )
);
```

mainAxisAlignment: MainAxisAlignment.center : Icon 가운데 정렬
