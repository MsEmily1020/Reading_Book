# Icon 떨어트려 가운데 정렬

```dart
MaterialApp(
  home : Scaffold(
    body: Row(
      mainAxisAlignment: MainAxisAlignment.spaceEvenly,
      children: [
        Icon(Icons.star),
        Icon(Icons.star),
        Icon(Icons.star),
      ],
    ),
  )
);
```

mainAxisAlignment: MainAxisAlignment.spaceEvenly : Icon 떨어트려 가운데 정렬
